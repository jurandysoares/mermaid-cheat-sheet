# Subgraphs


Syntax:

```markdown
subgraph title
    graph definition
end
```

Example:

```markdown
graph TB
    c1-->a2
    subgraph one
    a1-->a2
    end
    subgraph two
    b1-->b2
    end
    subgraph three
    c1-->c2
    end
```
